# Assets

Place textures, icons, sprite sheets, and future model/audio assets here.

## Placeholder Swap Contract

- During early development, placeholder images are allowed.
- Final art can replace placeholders with zero code changes if each file keeps the same:
  - relative path
  - file name
  - format
  - pixel dimensions
  - anchor convention (center-origin unless stated otherwise)

See `assets/visual/PLACEHOLDER_ASSET_LIST.md` for the current planned file map.
